у проекта есть installer. он находится в src/dist/RAG_Web_Installer.exe

для установки приложения достаточно просто запуцстить installer и далее у вас на рабочем столе появится exe файл с приложением.
так как мой проект работает полностью локально, то скорость установка и скорость генерации ответа rag зависит только от вашего железа
установка может долго стоять на середине - не пугайтесть, это скачивается локальная llm (llama3.1)
